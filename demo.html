Angular Timepicker Directive Demo

This directive creates a simple dropdown style timepicker, like the one used in Google Calendar.

Basic

Selected time: {{time | date:'h:mm a'}}

Code
<input type="text" dn-timepicker ng-model="time" />

Range

Selected time: {{time2 | date:'h:mm a'}}

Code
<input type="text" dn-timepicker ng-model="time" min-time="9:00" max-time="17:00" step="10m" />

Formatting

Selected time: {{time3 | date:'h:mm a'}}

Code
<input type="text" dn-timepicker ng-model="time" min-time="9:00" max-time="17:00" step="10m" time-format="HH:mm" />
